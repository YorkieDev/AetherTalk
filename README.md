### Quick Start

1. Clone this bad boy:
   ```
   git clone https://github.com/YorkieDev/AetherTalk
   ```

2. Slide into the project directory:
   ```
   cd AetherTalk
   ```
3. Open `index.html` in your browser and watch the magic happen!
